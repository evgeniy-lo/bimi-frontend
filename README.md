# Bimi - Frontend

Frontend part of bimi website.

## Installation

Use the [NPM](https://npmjs.com/) package manager to install all dependencies.

```bash
npm install
```

## Usage

Start project locally
```bash
npm run dev
```
Build production version of project
```bash
npm run build
```

Get archived version of production build
```bash
npm run zip
```
